About Me

My name is Nemanja Grujić, 23 years old, and I am a novice web developer. The idea to start doing this came to me one year ago when a friend told me about SoloLearn app, so I wanted to try it. I liked it very much because you get to experiment and see results right away. With that app I learned HTML and CSS basics, but it wasn't enough to do something bigger. I decided to quit. After few months I saw an add posted by Startit Centre Belgrade for HTML, CSS and JS course. I applied instantly. They took me in, and for couple months I have learned a lot. Today I can individualy make responsive web-sites and more.

I don't plan on stopping here. I want to learn more and one day become a great web developer.
